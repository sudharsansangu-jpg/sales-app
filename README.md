# Sales Management Backend

Express + MongoDB backend for the sales management application.

## Features

- JWT authentication
- Role-based access for `Admin`, `Manager`, and `Cashier`
- GST-aware invoice generation
- Customer balance tracking
- Dashboard and reports endpoints

## Setup

1. Copy `.env.example` to `.env`
2. Update MongoDB and JWT values
3. Run `npm install`
4. Run `npm run dev`

## Entry

- Server entry: `src/server.js`
